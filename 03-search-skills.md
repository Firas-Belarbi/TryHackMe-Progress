### 2. Google Search Operators
Learned how to use operators to refine search results:

| Operator | Function | Example |
|---------|----------|---------|
| `""` | Search exact phrase | `"error code 0x0000001"` |
| `site:` | Search inside one domain | `site:microsoft.com windows logs` |
| `filetype:` | Find specific file formats | `filetype:pdf incident response` |
| `intitle:` | Words inside page titles | `intitle:"login page"` |
| `-` | Exclude keywords | `phishing -email` |
| `OR` | Combine conditions | `linux OR ubuntu logs` |

These operators are extremely useful when analyzing alerts or researching malware.
